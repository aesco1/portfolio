# Portfolio

Personal portfolio website built with Next.js, React, Three.js, and Tailwind CSS.

## Live Demo

**[View Live Site](https://aesco1.github.io/portfolio)**

## Technologies

- Next.js 16
- React 19
- Three.js & React Three Fiber
- Framer Motion
- Tailwind CSS 4

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view in the browser.
